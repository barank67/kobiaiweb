# kobiaiweb

Kobi AI web sitesinin Astro ile hazırlanmış statik sürümü.

## Geliştirme

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

Build çıktısı `dist/` klasörüne üretilir.

## Icerik Ekleme

Blog ve basin yazisi ekleme kurallari icin [CONTENT_GUIDE.md](./CONTENT_GUIDE.md) dosyasini kullanin.
